### Install Hugo
```
sudo apt-get install hugo
```

### Clone repository

```
git clone 
```

### Install dependecies
``` 
cd blog
npm install
```

#### Watch changes (hugo/gulp)
```
npm run develop
```

#### Build site
```
hugo
```
